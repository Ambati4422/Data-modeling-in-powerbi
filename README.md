E-Commerce Sales Analysis using Power BI
📌 Project Overview
This project demonstrates the complete process of data cleaning, transformation, modeling, and analysis using Power BI. The analysis is performed on an e-commerce sales dataset to identify sales trends, profit performance, target achievement, and regional insights.
📂 Dataset Files
List of Orders.csv
Order Details.csv
Sales Target.csv
🔧 Data Transformation Tasks
✔️ Limited data to the first 500 records
✔️ Converted data types (Date, Amount, Target)
✔️ Formatted Customer Names into Proper Case
✔️ Created Location column by merging City and State
✔️ Calculated Profit Margin
✔️ Added Profit Status (Profit, Loss, Break-Even)
✔️ Handled Missing Values and Duplicates
🔗 Data Modeling
Established relationship between List of Orders and Order Details using Order ID.
Established relationship between Order Details and Sales Target using Category.
Verified active relationships in the data model.
🔄 Data Processing
Merged datasets into a consolidated Orders Data table.
Sorted orders by recent dates.
Filtered data for Tamil Nadu regional analysis.
Grouped and aggregated data for business insights.
📈 Analysis Performed
Order Count by Order ID
Average Profit by Category
Total Amount by Sub-Category
Monthly Sales Target Analysis
Regional Sales Analysis
Profitability Analysis
🛠️ Tools Used
Power BI
Power Query Editor
Data Modeling
Data Transformation
Data Cleaning
🎯 Key Insights
Improved data quality through cleaning and transformation.
Built an efficient data model using relationships.
Analyzed sales performance across categories and regions.
Compared actual sales against monthly targets.
Generated actionable business insights for decision-making.
